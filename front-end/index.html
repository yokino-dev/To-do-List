O site To Do List (Lista de Tarefas) serve para anotar informações de a fazeres do usuario de maneira organizada

Todo informação que o usuario coloca no site sera armanezado em um Servidor Local (Local Storage) que é hospedado no dispositivo do usuario que contenham acesso a pasta completa desse site

Os armanezamentos de dados são feito em JSON.

JSON é uma maneira leve e eficiente de armazenar dados, funcionando como uma espécie de “mini banco de dados” dentro do navegador ou no servidor.
